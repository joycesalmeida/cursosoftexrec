PROTOCOLO HTTP

A web funciona a partir do protocolo cliente-servidor, onde um cliente (como um navegador) faz solicitações a um servidor web e recebe respostas para exibir conteúdo na tela. Isso é feito usando o protocolo HTTP, onde o cliente envia uma solicitação HTTP para o servidor, que responde com um documento HTML e outros recursos.

CRIANDO UM PROJETO EM NODE.JS:

-- Usarei o npm express --

Após a instalação do node.js no computador, criei uma nova pasta para o projeto utilizando o terminal.
Para iniciar o projeto, executei o comando:

npm init

defini as configurações do projeto, após isso instalei o pacote npm express, com o seguinte comando:

npm install express

com esse framework podemos criar aplicativos web em Node.js
